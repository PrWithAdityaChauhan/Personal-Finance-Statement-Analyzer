# Personal Finance Analyzer

A Python-based finance analytics project that analyzes transaction data and provides spending insights through visual dashboards.

## Features

- Category-wise spending analysis
- Monthly spending trends
- Expense distribution visualization
- Top merchant analysis
- Spending leak detection

## Tech Stack

- Python
- Pandas
- Matplotlib

## Project Structure

├── data/

## Key Insights

- Spending categorized and analyzed.
- Monthly expenditure trends identified.
- Top merchants highlighted.
- Financial leak categories detected.
